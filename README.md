# TimerStopwatchApp
App that contains 2 functions: Timer and Stopwatch

Stack of technologies:
1. Jetpack Compose
2. Dagger\Hilt
3. MVVM
4. Coroutines
5. Multi-Moudles
6. DataStore
7. Compose Constraint Layout
8. Jetpack Compose Navigation
